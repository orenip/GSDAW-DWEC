Feedback 3 - Apartado 2

Solicita al usuario que introduzca una matrícula y utiliza expresiones regulares para validar que se ha introducido con el formato 1234ABC, es decir, 4 dígitos decimales seguidos de 3 letras. Avisa del error en caso de incumplimiento. Opcionalmente, puedes ampliarlo para que admita letras minúsculas y para que acepte también el formato 1234-ABC.
